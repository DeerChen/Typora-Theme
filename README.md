## Introduction

某自用魔改Typora主题。

## Features

* 继承`One Dark Pro`配色
* 更改标题字体为`优设标题黑`
* 更改代码块字体为`Dank Mono`
* 更改正文字体为`方正兰亭纤黑简体`
* 勾选框勾选之后会改变对应条目样式
* 统一侧边栏配色
* 修改表格配色

## Installation

放入主题文件夹：`%AppData%\Typora\themes`
